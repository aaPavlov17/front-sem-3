# By Aleks Shop — Clothing Retail Web App

## Описание приложения
By Aleks Shop — это современный интернет-магазин одежды, включающий удобный интерфейс для покупателей и интегрированного Telegram-бота для отслеживания заказов. Приложение позволяет просматривать каталог товаров, управлять корзиной, проходить аутентификацию и получать уведомления о статусе заказа в режиме реального времени.

## Как получить исходники
Для получения кода проекта выполните следующую команду в терминале:
```bash
git clone https://github.com/your-repository/front-sem-3.git
```

## Требования
Для работы приложения необходимы следующие компоненты:
- **Node.js**: версия 18 LTS или выше 
- **База данных**: SQLite 
- **Библиотеки**: Все зависимости устанавливаются автоматически через `npm install`

## Запуск проекта (среда *nix)

### 1. Установка зависимостей
Установите зависимости для фронтенда и бэкенда:
```bash
# Установка для фронтенда (корень проекта)
npm install

# Установка для бэкенда
cd backend && npm install
cd ..
```

### 2. Настройка переменных окружения
В файле .env в директории /backend лежит токен бота.


### 3. Запуск приложения
Запустите оба сервера (фронтенд и бэкенд) одновременно. В корне проекта выполните:

**Терминал 1 (Фронтенд):**
```bash
npm run dev
```
Фронтенд будет доступен по адресу: [http://localhost:5173](http://localhost:5173)

**Терминал 2 (Бэкенд):**
```bash
cd backend
npm run dev
```
Бэкенд сервер запустится на порту 3001.

## Главный бизнес-кейс
1. **Регистрация/Вход**: Перейдите на страницу регистрации и создайте новый аккаунт или войдите в существующий.
2. **Выбор товаров**: В каталоге на главной странице выберите понравившиеся товары и добавьте их в корзину.
3. **Оформление заказа**: Перейдите в корзину, проверьте список товаров и нажмите "Оформить заказ".
4. **Отслеживание**: После оформления вы увидите ссылку на Telegram-бота. Перейдите по ней и нажмите `/start`, чтобы привязать заказ и получать уведомления о его изменении (текщий заказ автоматически отправляется в бота)

## Структура приложения
- `src/components` — многоразовые UI компоненты.
- `src/pages` — основные страницы приложения.
- `src/store` — конфигурация Redux Toolkit.
- `backend/src` — исходный код бэкенда и логика Telegram-бота.
